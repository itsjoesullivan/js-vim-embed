#js-vim-embed

Embed js-vim on any web page

##Usage

```html
<!DOCYPE html>
<html>
<head>
</head>
<body>
  <div id='vim'></div>
  <script src='/vim.js'></script>
  <script>
    var vim = new Vim({el: document.getElementById('vim')});
  </script>
</body>
</html>
this code will help you in many ways so stay tuned
```
